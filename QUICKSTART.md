# QooCode 快速入门

## 🚀 30 秒快速开始

### 1. 安装插件
```
File → Settings → Plugins → Install Plugin from Disk...
选择 jetbrains-plugin-0.1.16.zip → OK → 重启 IDE
```

### 2. 配置 API
```
File → Settings → Tools → QooCode
填写 API Key、Base URL、Model
```

### 3. 开始使用
```
按 Ctrl + Shift + T 打开终端
输入：帮我分析这个项目
```

## 📋 配置示例

### DeepSeek (推荐)
```
API Key: sk-your-deepseek-key
Base URL: https://api.deepseek.com/v1
Model: deepseek-chat
```

### OpenAI
```
API Key: sk-your-openai-key
Base URL: (留空，使用默认)
Model: gpt-4o
```

### 通义千问
```
API Key: sk-your-aliyun-key
Base URL: https://dashscope.aliyuncs.com/compatible-mode/v1
Model: qwen-plus
```

## 💡 常用命令

```
分析项目结构        > 分析当前项目
创建文件            > 创建一个 User.tsx 文件
解释代码            > 解释这个函数
搜索代码            > 搜索所有 useEffect
Git 操作            > 查看最近的提交
运行测试            > 运行所有测试
```

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl + Shift + T | 打开 QooCode 终端 |
| Ctrl + Enter | 发送消息 |

## 🆘 遇到问题？

1. **找不到可执行文件**: 重启 IDE 并清理缓存
2. **API 密钥错误**: 检查 Settings → Tools → QooCode 配置
3. **更多帮助**: 查看 `jetbrains-plugin/使用指南.md`
