# QooCode JetBrains 插件使用指南

## 🚀 快速开始

### 1. 安装插件

1. 打开 IntelliJ IDEA
2. **File** → **Settings** → **Plugins**
3. 点击齿轮图标 → **Install Plugin from Disk...**
4. 选择 `jetbrains-plugin-0.1.16.zip`
5. 点击 **OK** 并重启 IDE

### 2. 配置 API 密钥

#### 方式一：通过 IDE 设置（推荐）

1. **File** → **Settings** → **Tools** → **QooCode**
2. 填写以下信息：
   - **API Key**: 你的 API 密钥
   - **Base URL**: API 地址（如 `https://api.deepseek.com/v1`）
   - **Model**: 模型名称（如 `deepseek-chat`）
3. 点击 **Apply** → **OK**

#### 方式二：通过环境变量

在系统环境变量中设置：
```bash
# Windows (PowerShell)
$env:OPENAI_API_KEY="your-api-key-here"
$env:OPENAI_BASE_URL="https://api.deepseek.com/v1"
$env:OPENAI_MODEL="deepseek-chat"

# Windows (CMD)
set OPENAI_API_KEY=your-api-key-here
set OPENAI_BASE_URL=https://api.deepseek.com/v1
set OPENAI_MODEL=deepseek-chat
```

### 3. 获取 API 密钥

| 服务商 | 获取地址 | Base URL | 推荐模型 |
|--------|---------|----------|---------|
| **DeepSeek** | https://platform.deepseek.com/api_keys | https://api.deepseek.com/v1 | deepseek-chat |
| **OpenAI** | https://platform.openai.com/api-keys | (默认) | gpt-4o, gpt-3.5-turbo |
| **通义千问** | https://dashscope.console.aliyun.com/apiKey | https://dashscope.aliyuncs.com/compatible-mode/v1 | qwen-plus |

## ⚙️ 高级配置

### 配置文件

在用户目录创建配置文件：
- **Windows**: `%USERPROFILE%\.qoocode\config.json`
- **Linux/macOS**: `~/.qoocode/config.json`

```json
{
  "apiKey": "your-api-key",
  "baseUrl": "https://api.deepseek.com/v1",
  "model": "deepseek-chat",
  "maxTokens": 8192,
  "temperature": 0.7,
  "timeoutMs": 120000,
  "debug": false,
  "verbose": false
}
```

### 配置优先级

命令行参数 > IDE 设置 > 环境变量 > 配置文件 > 默认值

## ❓ 常见问题

### Q: 提示 "Cannot find qoocode executable"？

**A:** 插件会自动从内部资源提取 qoocode.exe，无需额外安装。如果仍然报错，请：
1. 检查插件是否正确安装
2. 重启 IDE
3. 清理缓存：**File** → **Invalidate Caches** → **Restart**

### Q: 如何切换不同的 AI 模型？

**A:** 有三种方式：
1. 通过 **File** → **Settings** → **Tools** → **QooCode** 修改
2. 使用斜杠命令 `/model`
3. 修改环境变量 `OPENAI_MODEL`

### Q: 支持哪些 JetBrains IDE？

**A:** 支持所有基于 IntelliJ Platform 的 IDE：
- IntelliJ IDEA
- PyCharm
- WebStorm
- PhpStorm
- RubyMine
- GoLand
- CLion
- DataGrip

### Q: 如何自定义快捷键？

**A:** 
1. **File** → **Settings** → **Keymap**
2. 搜索 "QooCode"
3. 双击对应的动作进行自定义

### Q: 插件会收集我的代码吗？

**A:** 不会。插件在本地运行，代码只在调用 AI API 时发送给 AI 服务商，不会存储在其他地方。
